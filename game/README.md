# ひかりちゃんのゲームワールド 〜吸い込まれた世界〜

ゲームの世界に吸い込まれたひかりちゃんが、様々なミニゲームをクリアして元の世界に帰るブラウザゲームです。

## 遊び方

ブラウザで `index.html` を開くだけで遊べます。

### 操作方法
- **タップ/クリック** で操作（スマホ・PC対応）
- ステージ4のバイクレースは **長押し** で高くジャンプ

### ステージ一覧

| # | ステージ | 内容 | クリア条件 |
|---|---------|------|-----------|
| 1 | 💩 うんこ集め | 落ちてくるうんこをタップで集める | 100個集める |
| 2 | 🪱 ミミズ切り | ミミズをスワイプで切る（切ると分裂！） | 100匹切る |
| 3 | ⚔️ 包丁ゲーム | ガチャで鬼を決めて鬼ごっこ | 100回タッチ |
| 4 | 🏍️ バイクレース | 障害物を避けてゴールを目指す | ゴール到達 |
| 5 | 💄 変なメイク | キャラに変なメイクをする | 15人にメイク |
| 6 | 🚽 全部流す | ボタン連打で全部流す！ | 50回連打 |
| 👻 | お化けかくれんぼ | お化けから隠れてゴールへ | 隠しステージ |

### 隠しステージ
ステージ1〜6を全てクリアすると解放されます。

## 技術仕様
- HTML5 + CSS3 + JavaScript (ES Modules)
- Canvas API で描画
- Web Audio API でサウンド
- 外部ライブラリ不要
- レスポンシブ対応（スマホ・タブレット・PC）
- セーブデータは localStorage に保存
